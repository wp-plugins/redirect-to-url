== Description ==

This simple and lightweight WordPress Plugin Redirect To URL allows you to redirect your visitors to an external Website by clicking on an article or a page

**Features**

* Redirect Users to Websites
* Target website can be internal or external
* Link out of a page or an article
* No programming language needed
* Native designed box integrated in article and pages
* Unlimited redirections

== Installation ==

1. Install the Plugin
1. Activate the Plugin
1. Upon activation, you get a settings box in every article or page you create. Use this box to provide the address for redirection.
1. If the textbox is empty, redirection is disabled automatically. Activate it again by entering an address.
1. By visiting the article or page, the visitor will be redirected autmatically.
1. Thats it!
